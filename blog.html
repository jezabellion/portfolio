How to add a repository with GitKraken.

I love the GUI of GitKraken, but for someone using it as an entry point to using Git, or without prior Git knowledge, it has a super steep learning curve. To further the confusion, the GitKraken documentation is not exactly comprehensive. I ad significant diffuclty adding a repository to start off, and thought i’d share the correct process...

1. Create a remote Repository first (This is the main trick. It cannot be added locally after).

2. Include a readme file (it will not work without this)

4. Pull

5. Open GitKraken

6. Click on the Folder icon in the top left

7. Select “Clone” (this will clone a remote repo to your local computer)

8. Select the github.com icon - a drop down menu will appear

9. Select the desired local path

10. Clone repo! (Now any files you add locally can be pushed to Git.)

11. Add a new document via finder

12. Stage changes

13. Write a commit message

14. Commit

15. Push to Git!

Congratulations, you're done!
